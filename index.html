Getting Started

Before we get started, you need to add Magician Helpers to the source. Like if you're building a website.

<!DOCTYPE html>

<html lang="en">

  <head>

    <meta charset="utf-8" />

    <title>My First App</title>

  </head>

  <body>

    <h1>Hello, World!</h1>

    <script src="https://magician-helpers.netlify.app/index.js"></script>

    <script src="./index.js"></script>

  </body>

</html>

// JavaScript ES6

const { Helpers } = window;


// JavaScript ES5

var Helpers = window.Helpers;

Find more APIs in Magician Helpers documentation.
